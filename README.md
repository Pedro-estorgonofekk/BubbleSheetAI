# Bubble Sheet IA

## Como executar:

  * Instalar o python:
    
    * Instalando pelo [Link](https://www.python.org/downloads/)
  
## Comandos de execução:
  * Instalar as dependencias na raiz do projeto:

    * `pip install -r requirements.txt`

  * Mudar para a pasta da aplicação: 

    * `cd app`

  * Executar a aplicação:

    * `py main.py`

  * Acesse o arquivo `rotas.http` e solicite a requisição da criação do gabarito
